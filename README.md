EC513 Spring 2026 — Implementation of the YAGS (Yet Another Global Scheme) branch predictor in gem5. Evaluated on SPEC CPU2017 benchmarks (505.mcf_r, 520.omnetpp_r, 523.xalancbmk_r, 502.gcc_r, 525.x264_r) and compared against TournamentBP, LTAGE, and MultiperspectivePerceptronTAGE64KB
